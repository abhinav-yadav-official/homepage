# homepage

Static homepage for [abhiy.xyz](https://abhiy.xyz/).

## Files

- `index.html` - single-file portfolio page with inline CSS and a small typing-animation script.
- `Abhinav_Resume.pdf` - resume linked from the homepage.

## Local Preview

```sh
python3 -m http.server 8099
```

Then open `http://127.0.0.1:8099/`.
